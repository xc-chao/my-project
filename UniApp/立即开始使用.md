1. 打开 **微信开发者工具**
2. 点击 **"导入项目"**
3. 选择目录：`/Users/qq/Desktop/Uniapp/frontend`
4. AppID：选择 **"测试号"**
5. 点击 **"导入"**

### 步骤 2：配置环境（20 秒）

1. 点击右上角 **"详情"**
2. 切换到 **"本地设置"**

## 🆘 遇到问题？

### 后端启动失败

```bash
# 检查 MySQL 是否运行
mysql -u root -p

# 检查 Node.js 版本
node -v  # 应该 >= 16.0.0

# 重新安装依赖
cd backend
rm -rf node_modules
npm install
```

### 前端无法访问后端

1. 确保后端服务器已启动
2. 检查 `frontend/utils/request.js` 中的 BASE_URL
3. 微信开发者工具已勾选"不校验合法域名"

### 数据库连接失败

```bash
# 修改数据库密码
编辑 backend/src/config/database.js
将 DB_PASS 改为你的 MySQL 密码
```
