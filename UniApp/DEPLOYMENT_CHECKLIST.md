# ScanShop 部署验证清单

使用此清单确保项目正确部署。

## ✅ 环境准备

- [ ] Node.js >= 16.0.0 已安装
- [ ] MySQL >= 8.0 已安装并运行
- [ ] 微信开发者工具已安装

## ✅ 后端部署

### 依赖安装
- [ ] 执行 `cd backend && npm install`
- [ ] 无报错，所有依赖安装成功

### 环境配置
- [ ] 复制 `env.example` 为 `.env`
- [ ] 修改 `DB_PASS` 为你的 MySQL 密码
- [ ] 修改 `JWT_SECRET` 为随机字符串

### 数据库初始化
- [ ] 执行 `mysql -u root -p < database/schema.sql`
- [ ] 执行 `mysql -u root -p < database/seed.sql`
- [ ] 或执行 `npm run init-db`
- [ ] 数据库 `scanshop_db` 创建成功
- [ ] 4 张表创建成功（Users, Products, Orders, OrderItems）
- [ ] 测试数据插入成功

### 服务器启动
- [ ] 执行 `npm run dev`
- [ ] 看到提示：✅ 数据库连接成功
- [ ] 看到提示：🚀 服务器运行在 http://localhost:3000
- [ ] 浏览器访问 http://localhost:3000/api/products 返回 JSON 数据

## ✅ 前端部署

### 项目导入
- [ ] 打开微信开发者工具
- [ ] 导入 `frontend` 目录
- [ ] 填写 AppID（或选择测试号）

### 开发配置
- [ ] 详情 → 本地设置 → 勾选"不校验合法域名"
- [ ] 详情 → 本地设置 → 勾选"启用 ES6 转 ES5"
- [ ] 详情 → 本地设置 → 勾选"增强编译"

### 编译运行
- [ ] 点击"编译"按钮
- [ ] 无报错
- [ ] 模拟器显示小程序界面
