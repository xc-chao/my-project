# 论文绘图 Prompt 清单

下面内容按论文中的图表占位整理，可直接用于生成论文配图。所有图片统一采用下面这种风格，和数据库关系图示例保持一致：

- 白色背景，整体简洁专业，适合本科毕业论文
- 主体元素使用“蓝白表格卡片”样式：浅蓝色标题栏、白色内容区、蓝色边框、细蓝色分隔线
- 模块、实体、页面、流程节点都尽量画成表格卡片或蓝色边框矩形，不使用花哨图标
- 关系线使用浅灰色细线，可使用直线或平滑曲线，连接点使用小圆点
- 关系名称或流程说明使用浅黄色小标签标注，文字使用深灰色或黑色
- 字段、模块名称、流程节点文字必须清晰可读，避免乱码、水印、阴影和夸张透视
- 图片比例优先使用 16:9 或 A4 横向，保证放入 Word 后仍能看清文字
- 若生成页面截图类图片，也请生成“蓝白线框界面示意图”，不要生成真实照片风格界面

后续每一张图都默认继承以上样式；如果 prompt 中出现“风格要求”，也以本段统一风格为准。

- 订单 与 订单明细：1:N，一个订单包含多个订单明细
