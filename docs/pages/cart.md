# 购物车页

## 页面目标
集中展示已加购商品，支持调整数量、删除商品并进入确认订单。

## 入口与出口
- 入口：`tabBar` 购物车、商品详情页加购后回流。
- 出口：登录页、确认订单页、首页。

## 主要区域
- 顶部标题和商品数量统计。
- 购物车商品卡列表。
- 空状态卡片。
- 底部金额汇总和结算按钮。

## 交互说明
- 未登录时引导去登录。
- 已登录时可修改数量和移除商品。
- 点击“去结算”进入确认订单页。

## 字段/状态说明
- `list`：购物车行项目。
- `totalCount`：总件数。
- `totalAmount`：商品总金额。

## 对应接口
- `GET /cart`
- `POST /cart/items`
- `PATCH /cart/items/:id`
- `DELETE /cart/items/:id`

## 对应 Pencil 页面
- `CreatPencil.pen` 购物车页。
