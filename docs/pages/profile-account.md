# 个人信息管理页

## 页面目标

统一管理当前登录用户的基础资料，形成“登录 -> 查看我的 -> 修改资料/切换身份 -> 退出登录”的闭环。

## 入口与出口

- 入口：我的页头像、资料卡、账号与安全中的 `个人信息管理`。
- 出口：我的页、登录页。

## 权限规则

- 仅登录用户可进入。
- 未登录进入时展示登录引导。
- 身份切换当前用于演示环境，便于验证普通用户与管理员两套路径。

## 主要区域

- 顶部资料头图与当前头像预览。
- 基础资料区：昵称、手机号展示。
- 身份切换区：普通用户 / 管理员。
- 头像选择区：本地头像素材选择、从相册上传。
- 账号操作区：保存资料、退出当前登录。

## 交互说明

- 点击头像素材后，更新当前待保存头像。
- 点击 `从相册上传` 后，从系统相册选择图片并更新预览。
- 点击身份卡后，切换当前演示身份。
- 点击 `保存资料` 后，同步更新当前登录态。
- 点击 `退出当前登录` 后，清空当前登录信息并回到我的页未登录状态。

## 字段/状态说明

- `profile`：当前登录用户资料。
- `form.nickname`：可编辑昵称。
- `form.avatar`：当前选中的头像。
- `form.role`：当前选中的演示身份。

## 对应接口

- `GET /auth/profile`
- `PATCH /auth/profile`
- `POST /auth/logout`
