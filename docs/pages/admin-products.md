# 商品管理页

## 页面目标

供管理员查看商品上架状态、库存情况与销量表现，并执行新增、编辑、上下架、补库存、跳转前台详情核对等操作。

## 入口与出口

- 入口：后台信息管理页中的 `商品管理` 卡片。
- 出口：商品详情页、后台信息管理页。

## 权限规则

- 仅 `admin` 身份可访问。
- 普通用户不显示入口，手动访问也应被拦截。

## 主要区域

- 商品管理说明头部。
- 汇总卡片：商品总数、在售商品、低库存。
- 筛选栏：全部、在售、低库存、已下架。
- 商品管理列表。
- 商品编辑/新增表单页。

## 交互说明

- 点击 `新增商品`，进入商品表单页。
- 点击 `编辑`，进入当前商品的编辑表单页。
- 点击 `补库存`，对当前商品做库存补充。
- 点击商品卡，跳转前台商品详情页。
- 点击 `上架 / 下架`，切换当前商品销售状态。
- 商品表单支持使用预设图片、从相册选择封面/图集，并支持删除单张图集图片。
- 图集预览区支持拖拽调整顺序，第一张图默认作为详情首图。

## 字段/状态说明

- `products`：商品管理列表。
- `currentFilter`：当前筛选项。
- `saleStatus`：商品上下架状态。
- `stock`：库存数量。
- `editingId`：当前编辑商品 id。

## 对应接口

- `GET /api/admin/products`
- `GET /api/admin/products/:id`
- `POST /api/admin/products`
- `PATCH /api/admin/products/:id`
